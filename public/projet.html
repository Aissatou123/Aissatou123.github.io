Détection de maladies neurogénicales

Projet réalisé avc 3 autres étudiants. Ce projet consistait à utiliser nos compétences en Python et analyse de données afin de détecter des maladies neurogénicales grace à un code qui détecte la maladie d'alzheimer.

Détails du Projet

Ce projet a été réalisé dans le cadre de SDS3386. Il vise à détecter des maladies neurogénicales.

Technologies utilisées : python,regression logistique, analyse de données, panda, ...
